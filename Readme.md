# Pure Webpack

Dummy clean webpack config without any framework usage. Helps to understand how webpack works in general and why is it needed.


## Build and run

- Download depdendencies
```
npm install
```

- Run dev server
```
npm run start
```

- Dev build
```
npm run build-dev
```

- Prod build
```
npm run build
```

- Clear dist
```
npm run clear
```
